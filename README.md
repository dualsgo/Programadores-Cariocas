# 🚀 Bootcamp Programadores Cariocas - Explorer

Bem-vindo ao repositório central de materiais do **Bootcamp Programadores Cariocas**. Este repositório contém todos os slides, roteiros e materiais de apoio organizados por módulos.

---

## 🖥️ Painel Interativo (Dashboard)

Acabamos de adicionar um **Painel Premium** para facilitar a navegação por todos os arquivos do curso. Ele permite:
- 📂 **Navegação Organizada**: Explore pastas e subpastas de forma intuitiva.
- 🔍 **Busca Global**: Encontre qualquer slide ou PDF instantaneamente pelo nome.
- 🎨 **Interface Moderna**: Visual limpo, rápido e em modo escuro (glassmorphism).

### Como Visualizar o Painel Localmente

Como o painel utiliza módulos modernos de JavaScript, ele precisa ser servido através de um servidor local (HTTP) para funcionar corretamente no navegador.

1.  Abra o terminal na pasta raiz deste repositório.
2.  Execute o seguinte comando:
    ```bash
    npx serve .
    ```
3.  Acesse o endereço exibido (geralmente [http://localhost:3000](http://localhost:3000)).
